    # 316. 去除重复字母 (opens new window)

    这道题有三个要求:

    1. 去重 : 这个很好解决

    2. 顺序不变 : 这个也好解决

    3. 字典序最小 :

      这是什么意思呢?

      比如说上面的例子: 去重后得出的结果有 "bca" , "cab" , "abc"

      那么"abc"的字典序就是最小的

    利用单调栈的思想:

    将所求结果res=[]看作一个栈, 依次从s中取元素,

    1. 如果所取元素char>res[-1], 那么将char添加至res中.
    2. 如果char<=res[-1], 那么将res[-1]移除后再重复比较...

    这样得出的就是最基本的单调栈结果

    但是此题的要求是去重,且顺序不变, 所以需要有所修改:

    在char<=res[-1]时,要判断res[-1]在s剩下未取的元素中是否还有,如果没有了,则不移除res[-1]

    查看答案 
    class Solution:
    def removeDuplicateLetters(self, s: str) -> str:
        res = []
        for i in range(len(s)):
            char1 = s[i]
            if char1 in res:
                continue
            while res and char1 < res[-1] and res[-1] in s[i+1:]:
                res.pop()
            res.append(char1)
        return "".join(res)

    # 496. 下一个更大元素 I (opens new window)

    定义一个单调栈:

    每次向栈中放入元素, 若放入的元素大于栈底的,则移除栈底的元素, 再比较,一直到不再大于栈底的为止

    这样就会使该栈每次加入元素后,栈顶的永远是最大的,栈底是最小的, 而且栈底

    如: 有一个数组 nums2 = [1,3,4,2]

    注意是比某某大的下一个元素, 所以从右往左依次将元素加入栈中(如果是比某某大的上一个元素,则从左往右)

    将2与栈顶比较,发现栈底没有元素,则返回-1, 并将2放入栈中

    将4与栈底比较,发现4>2,则移除栈中的2,移除2,再比较,发现没有元素了,将4放入栈底,返回 -1

    将3与栈底的4比较,发现3<4,则返回4, 并将3放入栈底

    将1与栈底的3比较,发现1<3,则返回3, 并将1放入栈底

    查看答案 
    class Solution:
    def nextGreaterElement(self, nums1: List[int], nums2: List[int]) -> List[int]:
        res_dict = {}
        stack = []
        for i in range(len(nums2)-1,-1,-1):
            num = nums2[i]
            while stack and num > stack[-1]:
                stack.pop()
            res_dict[num] = stack[-1] if stack else -1
            stack.append(num)
        
        res = []
        for i in nums1:
            res.append(res_dict[i])
        return res

    # 739. 每日温度 (opens new window)

    查看答案 
    class Solution:
    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        res = [0] * len(temperatures)
        stack = []
        for i in range(len(temperatures)-1,-1,-1):
            temp = temperatures[i]
            while stack and stack[-1][0] <= temp:
                stack.pop()
            res[i] = stack[-1][1] - i if stack else 0
            stack.append([temp,i])
        return res
