pipenv管理虚拟环境

# pipenv管理虚拟环境

# 1. 安装pipenv
pip install pipenv
1
# 2. 创建虚拟环境
# 进入指定文件夹, 输入:
pipenv install
1
2

  1. 执行后会生成两个文件Pipfile,Pipfile.lock

  2. 想知道创建的虚拟环境在哪里,可以输入 pipenv --venv

    pycharm的解释器选择需要用到环境位置

# 3. 虚拟环境的使用
  • 方法一: 输入pipenv shell 激活虚拟环境后使用
  • 方法二: 使用 pipenv run + "命令"
# 4. 虚拟环境安装django
pipenv install django
1

这样安装在Pipfile中就会记录包信息 可以修改源,在Pipfile中的url换成"https://pypi.tuna.tsinghua.edu.cn/simple/"

# 5. 创建django项目
pipenv run django-admin startproject blogproject D:\blog-site
1

创建项目后加上路径, 就是在指定路径中创建项目(创建的manage.py与上文的两个文件Pipfile,Pipfile.lock在同一目录)
