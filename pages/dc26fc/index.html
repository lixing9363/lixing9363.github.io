    # 反转整个单链表----剑指 Offer 24. 反转链表 (opens new window)

    查看答案 
    # Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        pre = None
        cur = head
        while cur:
            nxt = cur.next
            cur.next = pre
            pre = cur
            cur = nxt
        return pre

    # 反转链表前 N 个节点

    将链表的前 n 个节点反转（n <= 链表长度）

    输入: 1-->2-->3-->4-->N , 3

    输出: 3-->2-->1-->4-->N

    笔记

    按照反转整个链表的方式,区别就在于第1步和最后判断迭代结束的第三步存在区别:

    • 在第1步中,将1节点指向2节点的指向打断,指向None应该变为指向第n+1节点
    • 其他的不变,还是反转
    • 到第3步中,当cur为None时,停止迭代变为当cur为第n+1节点时,停止迭代
    查看答案 
    class Solution:
    def reverseList(self, head: ListNode,n) -> ListNode:
        # 记录第n+1节点为n_and_1
        tmp = head
        for i in range(0,n):
            tmp = tmp.next
        n_and_1 = tmp
        # 迭代反转
        pre = None
        cur = head
        while cur.val != n_and_1.val:
            nxt = cur.next
            if pre:
                # pre不为None时,调换箭头方向
                cur.next = pre
            else:
                # pre为None时,说明是第一个节点,则需要指向n+1节点
                cur.next = n_and_1
            pre = cur
            cur = nxt
        return pre

    # 反转链表的一部分----92. 反转链表 II (opens new window)

    分析

    此题与反转前N个节点的区别就在于left是否为1:

    • 如果left为1,则就是反转前right个节点,无区别
    • 如果left不是1, 则有所区别:
      1. 当cur为left时, 要指向right+1节点
      2. left-1节点要指向right节点
    查看答案 
    # Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseBetween(self, head: ListNode, left: int, right: int) -> ListNode:
        # 找到初始pre节点(left-1节点)
        # 找到cur节点
        if left == 1:
            pre = None
            cur = head
        else:
            pre = head
            for i in range(left - 2):
                pre = pre.next
            cur = pre.next
        
        # 找到right节点,right+1节点
        right_node = head
        for i in range(right-1):
            right_node = right_node.next
        right1_node = right_node.next

        pre_tmp = pre
        while cur != right1_node:
            nxt = cur.next
            if pre_tmp:
                if pre_tmp != pre:
                    cur.next = pre_tmp
                else:
                    cur.next = right1_node
                    pre_tmp.next = right_node
            else:
                cur.next = right1_node
            pre_tmp = cur
            cur = nxt
        if pre:
            return head
        return right_node

    # 25. K 个一组翻转链表 (opens new window)

    分析

    1. 首先, 可以肯定, 一组一组的翻转可以使用的方法就是迭代或者递归
    2. 每一组翻转后, 翻转后的尾节点,指向右边翻转后的头节点

    那么:

    我们可以发现,因为每次翻转后,尾节点要指向右边翻转的头节点,所以要从后面向前迭代执行翻转.

    所以我们可以参考逆序递归

    查看答案 
    # Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseKGroup(self, head: ListNode, k: int) -> ListNode:
        a = b = head
        for _ in range(k):
            if not b:
                return head
            b = b.next
        new_head , new_tail = self.reverse(a,b)
        new_tail.next = self.reverseKGroup(b,k)
        return new_head


    
    def reverse(self,a,b):
        # 将节点a,b进行翻转,返回翻转后的头节点和尾节点
        pre = None
        cur = a
        while cur != b:
            nxt = cur.next
            cur.next = pre
            pre = cur
            cur = nxt
        return pre,a

    # 234. 回文链表 (opens new window)

    翻转后半部分,再比较

    查看答案 
    # Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def isPalindrome(self, head: ListNode) -> bool:
        fast = slow = head
        while fast and fast.next:
            fast = fast.next.next
            slow = slow.next
        reversed_head = self.reverse(slow)
        while reversed_head:
            if reversed_head.val != head.val:
                return False
            reversed_head = reversed_head.next
            head = head.next
        return True
    
    def reverse(self,head):
        pre = None
        cur = head
        while cur:
            nxt = cur.next
            cur.next = pre
            pre = cur
            cur = nxt
        return pre
