    # 14. 最长公共前缀 (opens new window)

    提示

    1. 找到最短的字符串
    2. 用一个指针当作游标分别与最短的字符串对应指针处做比较，如果不相同，则得到最长公共前缀
    # 解答：
    查看答案 
    class Solution:
    def longestCommonPrefix(self, strs: List[str]) -> str:
        min_str = strs[0]
        for i in strs:
            if len(i) < len(min_str):
                min_str = i
        
        rs = ""
        for i in range(len(min_str)):
            char1 = min_str[i]
            flag = False
            for str1 in strs:
                if str1[i] != char1:
                    flag = True
                    break
            if flag:
                return rs
            rs=min_str[:i+1]
        return rs
