Django----分页

# Django----分页

# Paginator对象

  • 构造对象
Paginator(object_list, per_page)
#object_list =>数据列表
#per_page =>每页数据个数
1
2
3
  • 对象属性
    • count:数据总数
    • num_pages：分页后的页面总数
    • page_range：当前页面码数,从1开始
    • per_page :每页数据的个数

# Page对象

  • 对象构造:通过Paginator对象构造
page = paginator.page(cur_page)
#cur_page => 当前页码(int)
#page对象是可迭代对象,迭代出来的每个数据就是单个的数据
1
2
3
  • 对象属性
    • has_next()：如果有下一页返回True
    • has_previous()：如果有上一页返回True
    • has_other_pages()：如果有上一页或下一页返回True
    • next_page_number()：返回下一页的页码，如果下一页不存在，抛出InvalidPage异常
    • previous_page_number()：返回上一页的页码，如果上一页不存在，抛出InvalidPage异常
    • len()：返回当前页面对象的个数

  • 通过templates渲染页面时:

    def test_page(request):
    # /index/test_page?page=3
    all_book = Book.objects.all()

    page = paginator.page(cur_page)

  • 前后端分离时:

    前后端分离时所有数据都是使用json传输,所以需要将对象转化为json数据

    def address_page(request):
    if request.method == 'GET':
        user = UserProfile.objects.get(id=1)
        cur_page = request.GET.get('cur_page', 1)
        all_address = Adress.objects.filter(user=user,isActive=True)
        paginator = Paginator(all_address, 2)
        page = paginator.page(cur_page)
        address_list = []
        for address in page:
            address_dict = {"address_name": address.address_name}
            address_dict["detail_address"] = address.detail_address
            address_dict["consignee"] = address.consignee
            address_dict["province"] = address.province
            address_dict["city"] = address.city
            address_dict["area"] = address.area
            address_dict["id"] = address.id
            address_dict["isDefault"] = address.isDefault
            address_list.append(address_dict)
        total_pages = paginator.num_pages
        data = {'address_list': address_list, 'total_page': total_pages}
        resp = {'status': 200, 'data': data}
        return JsonResponse(resp)
