# 300. 最长递增子序列

# 一.动态规划解法

# 思路

  • 动态规划解决问题,最重要的是找到状态转移方程,就是将一个大的问题怎么逐步分解为另一个.

    定义一个数组dp:

    dp[i]表示以 nums[i] 这个数结尾的最长递增子序列的长度

    举例:

    nums = [4, 10, 4, 3, 8, 9] i = [0, 1, 2, 3, 4, 5]

    dp数组 dp(i)与{dp(0)...dp(i-1)}关系 dp(i)与{dp(0)...dp(i-1)}组合 升序子列表长度 最终长度
    dp(0) [4] 1 1
    dp(1) dp(0) 与 nums[1] [4]与[10] -->[4,10] 2 2
    与自己单独 [10]与[10]-->[10] 1
    dp(2) dp(0) 与 nums[2] [4]与[4] -->[4] 1 1
    dp(1) 与 nums[2] [4,10] 与[4]-->[] 0
    与自己单独 [4]与[4]-->[4] 1
    dp(3) dp(0) 与 nums[3] [4]与[3] -->[4,3] 0 1
    dp(1) 与 nums[3] [4,10]与[3] -->[4,10,3] 0
    dp(2) 与 nums[3] [4]与[3] -->[4,3] 0
    与自己单独 [3]与[3]-->[3] 1
    dp(4) dp(0) 与 nums[4] [4]与[8] -->[4,8] 2 2
    dp(1) 与 nums[4] [4,10]与[8] -->[4,10,8] 0
    dp(2) 与 nums[4] [4]与[8] -->[4,8] 2
    dp(3) 与 nums[4] [3]与[8] -->[3,8] 2
    与自己单独 [8]与[3]-->[8] 1
    dp(5) dp(0) 与 nums[5] [4]与[9] -->[4,9] 2 3
    dp(1) 与 nums[5] [4,10]与[9] -->[4,10,9] 0
    dp(2) 与 nums[5] [4]与[9] -->[4,9] 2
    dp(3) 与 nums[5] [3]与[9] -->[3,9] 2
    dp(4) 与 nums[5] [4,8]与[9] -->[4,8,9] 3
    与自己单独 [9]与[9]-->[9] 1

    dp=[1,2,1,1,2,3]

    可以看到最后结果为3,[4,8,9]

    从上述表格可以看出dp(i)与{dp(0)...dp(i-1)}规律:

    1. 要想通过{dp(0)...dp(i-1)}得出dp(i),则需要用nums[i]分别与{nums[0]...nums[i-1]}比较
    2. 如果nums[i]大,则相当于将{dp(0)...dp(i-1)}的最小子序列再往后添加一个,所以长度+1
    3. 如果nums[i]小或者相等,则说明,不能与{dp(0)...dp(i-1)}的最小子序列组合起来再形成升序序列,所以长度为0,但是可以就自己单独一个元素为序列,形成一个长度为1的升序序列,所以长度为1
    4. 如上表的dp(5)的比较后的长度[2,0,2,2,3,1],最长的以nums[5]=9结尾的子序列长度是3
    5. 得到dp(5) = 3

    最终结果就是取dp数组的最大值

# 代码
class Solution:
    def lengthOfLIS(self, nums) -> int:
        # 初始化备忘录,dp_memo[i]表示以 nums[i] 这个数结尾的最长递增子序列的长度
        dp_memo = [0] * len(nums)
        if len(nums) == 1:
            # 如果nums长度为1,则最小子序列长度就是1
            dp_memo[0] = 1
        else:
            for i in range(0, len(nums)):
                # dp_memo[n] 分别与 dp_memo[0]...dp_memo[n-1]比较后的值添加到tmp,最起码长度为1,所以初始化[1]
                tmp = [1]
                for j in range(0, i):
                    if nums[i] > nums[j]:
                        # 如果后面比前面大,则肯定长度加1
                        tmp.append(dp_memo[j] + 1)
                    else:
                        # 否则,不能组成升序数组,但是最起码长度是1
                        tmp.append(1)
                dp_memo[i] = max(tmp)
        return max(dp_memo)

# 优化后代码

1.因为不管如何,最小升序子序列长度起码为1,所以初始化备忘录的值应该是1

2.起码长度为1后,就不需要判断nums长度是否是1的情况了

3.且不需要中间列表tmp来取最大值,只需要每次比较时与对应的dp_memo[i]相比较取最大值即可

class Solution:
    def lengthOfLIS(self, nums) -> int:
        # 初始化备忘录,dp_memo[i]表示以 nums[i] 这个数结尾的最长递增子序列的长度
        dp_memo = [1] * len(nums)
        for i in range(0, len(nums)):
            for j in range(0, i):
                if nums[i] > nums[j]:
                    # 如果后面比前面大,则长度加1,且要与当前时间的dp_memo[i]相比较,取大的
                    dp_memo[i] = max(dp_memo[i], dp_memo[j] + 1)
        return max(dp_memo)
